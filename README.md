# Robolearn

Plataforma web para el aprendizaje de programación y robótica para niños y adolescentes.

El siguiente proyecto cuenta con arquitectura hexagonal. Las tecnologias para el frontend son svelte, para el backend usamos FastAPI y para el chatbot estamos con dialogflow.

Desplegar con docker

```bash
docker compose up --build
```
